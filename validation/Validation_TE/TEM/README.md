This folder is for the mesh convergence study in the first half of Section 4.1 of the paper by Guillen and Charlot.


NOTE: The files in the current directory that are NOT organized into other folders are for the *medium mesh size*.


**run_power.py**
This Python script runs the *module_cuboid_pattern_sq.i* input file at varying currents from I = 1.6 A to I = 2.5 A and saves the outputs to files called pattern_#.csv and pattern_#.e
NOTE: *run_power.py* and *run_power_font.py* are exactly the same.


**plot_power.py**
This Python script plots the Power vs. Current from the pattern_#.csv files generated by the *run_power.py* Python script. A figure titled TEM_power.png is outputted.
NOTE: *plot_power.py* and *plot_power_font.py* are exactly the same.


**plot_meshes_font.py**
This Python script compares the Power vs. Current plots for all three different mesh refinements: coarse, medium, and fine. The outputs are figures at two different resolutions: TEM_power_mesh_font.png and TEM_power_mesh_zoom_font.png.
NOTE: *plot_meshes.py* is different from the aforementioned Python script because it uses the MOOSE Python library: thm_utilities.
